# Planificateur de voyages

## Description
Site d'aide à la préparation et à l'organistation de voyages.
Contient des informations utiles à l'élaboration d'itinéraires et à l'estimation de budgets. 

## Outils et langages
- CMS Symfony
- HTML, CSS, Sass, Javascript
